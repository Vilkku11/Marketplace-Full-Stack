[Frontend](https://marketplace-final-front.onrender.com)

## Installation and local deployment

At the root of project folder:
`docker compose up -d`

Start backend:
`cd backend`
`npm install`
`node server.js`

Start frontend
`cd frontend`
`npm install`
`npm run dev`
